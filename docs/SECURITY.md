# 安全基线

## 身份与鉴权

- 学生端默认通过微信小程序登录建立会话，再完成教务身份绑定
- 登录支持微信身份与教务身份绑定
- Access Token 与 Refresh Token 分离
- 管理员账号支持更严格的会话策略
- 联系方式等受限字段仅对已完成教务绑定的账号开放

## 权限控制

- 基于 `RBAC + 资源动作码`
- 关键管理操作强制审计
- 数据权限与组织范围绑定
- 联系方式查看权限必须由后端实时校验，不能仅依赖前端隐藏

## 数据保护

- 敏感字段最小化采集
- 手机号、身份证号等敏感信息按需脱敏展示
- 文件上传校验类型、大小、病毒风险

## 配置与密钥

- 密钥不入库、不入仓库
- 统一通过环境变量或密钥管理系统注入

## 审计

- 登录
- 教务绑定
- 联系方式查看
- 权限变更
- 内容发布
- 内容审核
- 外部系统同步失败重试
